## Frontend Skills

### Next.js (App Router)
- Build applications using the App Router architecture
- Implement file-based routing and layouts
- Use Server Components by default for performance
- Handle metadata, loading, and error states correctly

### TypeScript
- Apply static typing to React components
- Define clear interfaces for API responses
- Prevent runtime errors through type safety
- Improve maintainability and refactoring confidence

### React Server Components
- Separate server and client responsibilities
- Fetch data securely on the server
- Minimize client-side JavaScript
- Optimize rendering performance

### Tailwind CSS
- Create responsive and consistent UI layouts
- Use utility-first styling patterns
- Maintain design consistency across components
- Avoid custom CSS unless necessary

### API Integration Patterns
- Consume REST APIs using fetch or API clients
- Pass JWT tokens securely in request headers
- Handle loading, error, and success states
- Keep API logic separate from UI components
