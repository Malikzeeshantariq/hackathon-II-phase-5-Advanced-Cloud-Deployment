## Database Skills

### PostgreSQL
- Design and manage relational schemas
- Write efficient SQL queries
- Enforce data integrity and constraints
- Work with serverless PostgreSQL (Neon)

### SQLModel
- Define ORM models mapped to PostgreSQL tables
- Manage relationships and field constraints
- Integrate SQLModel with FastAPI sessions
- Ensure schema consistency between code and specs

### Data Modeling
- Translate business requirements into normalized tables
- Design user-scoped data ownership models
- Choose appropriate data types and constraints
- Avoid redundancy while maintaining clarity

### Indexing & Performance Basics
- Identify frequently queried fields
- Create indexes for user_id and lookup columns
- Understand query execution trade-offs
- Avoid premature optimization

### Schema Versioning Concepts
- Track schema changes via specifications
- Maintain backward compatibility where possible
- Plan additive changes safely
- Align schema evolution with application phases
